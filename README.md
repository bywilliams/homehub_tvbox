# HomeHub

HomeHub é um Gateway de Automação Residencial desenvolvido para reutilizar hardware de baixo custo, como TV Boxes Android, transformando-os em servidores domésticos para automação, armazenamento de arquivos e integração com dispositivos IoT.

## Hardware Atual

- TV Box RK3066
- Android 11
- Termux
- 1 GB RAM
- 8 GB armazenamento
- Cartão SD 256 GB

## Objetivos

- Gateway MQTT
- API REST
- Servidor de Arquivos
- Dashboard Web
- Integração com ESP32
- Automação Residencial

Projeto em desenvolvimento.
